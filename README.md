**A PyQt6-based GUI for mice behavior scoring**

This tool provides frame-by-frame video navigation and manual logging of behavioral events.

**Features**

    Video Navigation: Supports frame-by-frame playback.
   
    Record Editing: Allows for post-entry adjustments to existing behavioral records.
    
    Manual Logging: Drag-and-drop record logging for behavioral events.

    Configuration: Allows for customizable ethogram settings.

    Data Handling: Standardized import and export of scoring data.

**Requirements**

    Python 3.x

    PyQt6

    OpenCV

**Usage**

    Run the script: python behavior_scorer.py.

    Load a video file into the application.

    Log behaviors and edit records as required.

    Export your logs for research analysis.
